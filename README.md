# RayTracer
A ray tracer in C++

## Dependencies
- libpng++-dev >= 2.10  
  install with apt: `sudo apt instal libpng++-dev`


## How to build and use

### Compilation
- build project: `make`
- build examples: `make examples`
- build documentation: `make doc`
- clean binaries: `make clean`

### Run
`./RayTracer`
